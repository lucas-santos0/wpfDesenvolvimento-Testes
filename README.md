# 🔢 Conjuntos – Operações Matemáticas (WPF)

Projeto desenvolvido para a disciplina de **QTS**, com o objetivo de implementar um sistema WPF capaz de realizar operações matemáticas entre os conjuntos A e B.  
O software permite adicionar valores, gerar conjuntos aleatórios e executar operações clássicas da teoria dos conjuntos.

Além disso, o projeto inclui uma **tela oculta de testes**, acessada através do comando `/tpOn`, criada para validar as funções internas do sistema.

---

## 🚀 Funcionalidades

### 🟦 Conjunto A e Conjunto B
- Adicionar valores  
- Remover valores  
- Gerar valores aleatórios  
- Limpar conjuntos  

### 🧮 Operações Principais entre Conjuntos
- União  
- Interseção  
- Diferença  
- Verificação de Subconjunto  
- Exibição dos valores  

### 🧪 Tela Especial de Testes (modo secreto)
A tela de testes foi criada para executar comandos e validar as operações internas.  
Ela **não aparece normalmente** — só é ativada pelo comando `/tpOn`.

Nessa tela é possível realizar todas as operações da interface principal, além de testar e validar o funcionamento interno das funções do sistema.

## ⚙️ Como Executar
1. Abra o projeto no **Visual Studio**  
2. Execute a aplicação (**F5**)  
3. Use a interface para adicionar valores e realizar operações  
4. Para abrir a tela secreta de testes, digite: `/tpOn`
